# Chittr

A Twitter clone in Ruby and Sinatra.

## The stack

* Ruby
* Sinatra
* Postgres and DataMapper
* Twitter Bootstrap

For testing:
* RSpec
* Cucumber/Gherkin
* Capybara

Running on Heroku.
